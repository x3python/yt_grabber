# GUI for grabbing mp3 files from youtube and many others places powered by tkinter and framework yt_dlp.

Requirements:
- python3
- yt_dlp ... that's it.

1. git clone https://github.com/x3python/yt_grabber
2. cd yt_grabber
3. python3 yt_grabber.py

Enjoy your grabbed music!
